# 🌩️ Pathboard

**Pathboard** is a clean, student-focused productivity app designed to help you turn overwhelming deadlines into structured, manageable journeys.

Instead of juggling multiple tools for tasks, notes, and planning, Pathboard brings everything into one place — simple, visual, and aesthetic.

---

## ✨ Why Pathboard?

Most productivity tools are either:

* Too complex (too many features you don’t need)
* Too generic (just another to-do list)

Pathboard focuses on what students actually need:

> **Clear structure + visual progress + simple note-taking**

---

## 🚀 Core Features

### 🧩 Learning Paths (Projects)

Break down big goals into smaller, actionable steps.

* Create paths for exams, assignments, or projects
* Add step-by-step tasks
* Track progress visually
* Set due dates and stay on track

---

### 📊 Smart Tile Board

Your projects are displayed as **interactive tiles**:

* Drag and organize freely
* Merge tiles into folders (like mobile apps)
* See progress at a glance
* Urgency indicators (colors show what needs attention)

---

### 🔥 Urgency System

Stay focused on what matters most.

* 🔴 Due today / overdue
* 🟠 Due soon
* 🟡 Coming up
* ⚪ Not urgent

No need to overthink priorities — Pathboard highlights them for you.

---

### 📝 Markdown Notes

Write and organize your notes directly inside the app.

* Clean markdown editor
* Toggle preview mode
* Supports:

  * Headers
  * Lists
  * Code blocks
  * Formatting

Example:

```md
# Algorithms Midterm

## Topics
- Graphs
- Dynamic Programming

## Important
Dijkstra: O(E log V)
```

---

### 💾 Local File Saving (No Accounts Required)

Your notes are **yours**.

* Press **Ctrl + S** to save notes
* Choose where to store them on your PC
* Files are saved as `.md` (markdown)
* No cloud, no login, no subscriptions

---

### ⚙️ Simple Settings

* Toggle dark mode
* Enable/disable features
* Keep the app minimal and distraction-free

---

## 🎯 Philosophy

Pathboard is built around one idea:

> **Make studying feel structured, not overwhelming.**

Instead of:

* Endless lists
* Scattered notes
* Forgotten deadlines

You get:

* A clear path
* A visual overview
* A calm, focused workflow
